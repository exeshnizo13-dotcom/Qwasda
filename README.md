# Qwasda 🇺🇦🇬🇧

Автоматичне визначення та перемикання розкладки клавіатури між українською та англійською.

## Можливості

- ⌨️ Ручне перемикання поточного слова — **подвійний Shift**
- 🔄 Автокорекція розкладки за словниками (укр ↔ англ)
- 🖥 Системний трей з меню керування
- ▶️ Автозапуск при старті Windows
- ⏸ Вмикання/вимикання перемикання через трей

## Як це працює

1. Програма працює у фоні та перехоплює натискання клавіш.
2. **Ручне перемикання:** двічі натисніть **Shift** (без іншої клавіші між ними) —
   останнє набране слово конвертується між розкладками.
3. **Автокорекція:** на межі слова (пробіл/Enter/Tab) програма звіряє слово зі
   словниками. Якщо слово невалідне в поточній мові, але його конвертація в іншу
   розкладку дає валідне слово — програма виправляє розкладку:
   - видаляє помилково набране слово,
   - друкує правильний варіант,
   - перемикає розкладку активного вікна.

   Наприклад, `ghbdtn` у англійській розкладці → `привет` (а `ghbdsn` → `привіт`):
   латинські клавіші відповідають кириличним позиціям ЙЦУКЕН.

> **Словники.** Автокорекція використовує `data/words_en.txt.gz` та
> `data/words_uk.txt.gz` (частотні/повні списки слів). Їх можна замінити власними
> gzip-файлами у форматі «одне слово на рядок». Без словників працює лише ручне
> перемикання подвійним Shift.

## Вимоги

- Windows 10/11
- Python 3.8+

## Швидкий старт

### Варіант 1: Через .bat файл (рекомендовано)

Просто запустіть **`start.bat`** — він автоматично встановить залежності та запустить програму.

### Варіант 2: Вручну

```powershell
pip install -r requirements.txt
python qwasda.py
```

## Налаштування автозапуску

### Через setup.bat (рекомендовано)

Запустіть **`setup.bat`** — він встановить залежності та додає Qwasda до автозапуску Windows.

### Через трей

Правий клік на іконці в треї → "Автозапуск увімкнено/вимкнено"

## Керування через трей

Правий клік на іконці в системному треї:

- **Перемикання увімкнено/вимкнено** — увімкнути або призупинити автоперемикання
- **Автозапуск увімкнено/вимкнено** — додати або видалити з автозапуску
- **Вихід** — зупинити програму
