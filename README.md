# Qwasda 🇺🇦🇬🇧

Автоматичне визначення та перемикання розкладки клавіатури між українською та англійською.

## Як це працює

1. Програма працює у фоні та перехоплює натискання клавіш
2. Коли ви друкуєте текст у неправильній розкладці (наприклад, `ghbdtn` замість `привіт`), програма:
   - Видаляє помилково набраний символ
   - Перемикає розкладку
   - Вводить правильний символ

## Вимоги

- Windows 10/11
- Python 3.8+

## Встановлення

```powershell
pip install keyboard
```

## Запуск

```powershell
python qwasda.py
```

## Вихід

Натисніть `Ctrl+C` у терміналі.

## Приклади

| Набрано (помилково) | Розкладка | Результат |
|---------------------|-----------|-----------|
| `ghbdtn`            | EN        | `привіт`  |
| `руддщ`             | UK        | `hello`   |
